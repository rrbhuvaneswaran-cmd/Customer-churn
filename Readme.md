# Customer Churn Prediction System

## Project Overview
This project builds a Machine Learning model to predict whether a telecom customer will leave the company (churn) based on historical customer data.

Customer churn prediction helps companies identify customers who are likely to leave and take actions to retain them.

## Dataset
The dataset contains customer information such as:

- Age
- Gender
- Tenure
- Monthly Charges
- Contract Type
- Payment Method
- Total Charges
- Churn Status

The dataset is used to analyze customer behavior and build predictive models.

The following techniques were used in this project:

## Model Performance
The models were evaluated using:

- Accuracy Score
- Confusion Matrix
- Classification Report

Random Forest achieved better performance compared to Logistic Regression.

## Key Insights
- Customers with higher monthly charges are more likely to churn.
- Customers with shorter tenure have a higher churn probability.
- Long-term contract customers are less likely to churn.


## Business Impact
This system can help companies:

- Identify high-risk customers
- Improve retention strategies
- Reduce customer churn
- Increase business revenue
